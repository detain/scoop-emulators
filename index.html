scoop-emulators - Latest Emulators and Versions

scoop-emulators is a constantly updating list of the latest emulators and versions available in the Scoop package manager for Windows. This repository aims to provide easy installation and management of various emulators for gaming and other purposes.

Below you'll find the list of currently supported emulators along with their respective versions:

Emulator Version
SNES9x v1.60
Dolphin v5.0
PCSX2 v1.6.0

To install any of these emulators using Scoop, you can run the following command in your command-line interface: 

scoop install <emulator_name>

For more details on Scoop and how to use it, please visit the official Scoop website.
